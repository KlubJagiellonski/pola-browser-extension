Pola Web Extension
==================

Rozszerzenie przeglądarki Chrome do aplikacji `Pola <https://www.pola-app.pl/>`_.
Skanuje strony internetowe w poszukiwaniu kodów EAN/GTIN produktów i wyświetla informacje
o polskości firmy korzystając z API Pola (v4).

Install
=======

Chrome
------

1) Wpisz ``chrome://extensions/`` w pasek adresu i naciśnij **ENTER**
2) Włącz **Developer mode** (prawy górny róg)
3) Kliknij **Load unpacked**
4) Wskaż folder z rozszerzeniem i kliknij **OK**
